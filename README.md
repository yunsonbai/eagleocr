# eagleocr
## Overview
* 应用：识别图片中的文字
* 版本：0.1
* 安装：
	* git clone git@github.com:yunsonbai/eagleocr.git
	* python setup.py install
* 已经实现的功能:
	* 文字切割

* 还需完成的功能：
    * 文字学习
    * 文字识别
    * 图片矫正

## Requirements
* 开发语言
	* python3.5
* 依赖库
    * PIL==4.1.1
    * opencv-python==3.2.0.7
    * numpy==1.13.0
